# Skill: GDN/PMax Audit + Suggest Banner

## Trần CPA Display
**Chặt 0.5x** Giá bán − Vốn − VAT (chặt hơn search vì traffic chất lượng thấp hơn)

## 6 nhóm chấm điểm GDN

| Nhóm | Trọng số |
|------|----------|
| Asset (banner/video count + format) | 25% |
| PMax Rating | 20% |
| Placement (chất lượng site) | 20% |
| Targeting | 15% |
| Measurement | 10% |
| Cấu trúc | 10% |

## 5 cơ chế suggest banner

1. **TOP PERFORMER REPLICATION** — banner thắng → biến thể size khác
2. **REPLACE LOSER** — brief đầy đủ thay banner CTR thấp
3. **FILL ASPECT RATIO GAP** — thiếu format nào → bổ sung
4. **A/B CHALLENGER** — challenger cho winner hiện tại
5. **SEASONAL** — theme mùa/sự kiện

## Brief banner format

| Trường | Mô tả |
|--------|-------|
| Size | 1200×628, 1080×1080, 1080×1350, 1080×1920 |
| Layout | 4 vùng: hero / headline / USP / CTA |
| Color hex | Brand color: Doscom #1e3a8a, NOMA #1f2937 |
| Copy | Headline ≤30 chars, USP ≤90 chars |
| CTA | "Mua ngay", "Xem thêm", "Tư vấn" |
| Hypothesis | "Banner X sẽ thắng vì Y" |
| Cost | AI tự dựng ~80K, designer ~300K |
