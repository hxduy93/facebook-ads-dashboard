# Skill: Doscom & NOMA Brand Guidelines

## Brand Voice — Doscom

- **Định vị**: Hệ sinh thái công nghệ đa lĩnh vực bảo vệ riêng tư
- **Tone**: Chuyên nghiệp, đáng tin cậy, kỹ thuật rõ ràng (không cường điệu)
- **Audience chính**: An ninh cá nhân + DN/VIP + người yêu xe (NOMA)

## Brand Voice — NOMA

- **Định vị**: DIY Auto Care chuẩn Mỹ, premium đại chúng
- **Tone**: Nam tính, hành động, hiệu quả ngay
- **Audience**: Nam 25-45, sở hữu ô tô, thích DIY

## Typography

- **Heading**: Segoe UI Bold / Inter Bold / Montserrat
- **Body**: Segoe UI / Inter / Roboto
- **Mono (code/data)**: JetBrains Mono / Consolas

## Cấu trúc tài liệu chuẩn

### Quy định phòng ban

- Chương I: Quy định chung
- Chương II: Chức năng và Nhiệm vụ
- Chương III: Trách nhiệm cụ thể
- Chương IV: Quản trị vận hành
- Chương V: Tổ chức thực hiện

### Báo cáo tuần (PPTX)

1. Slide bìa (theme đen/vàng)
2. Tóm tắt 1 slide (top 3 finding)
3. Số liệu chính (CPL, CIR theo 5 SP)
4. So sánh tuần
5. Action tuần tới
