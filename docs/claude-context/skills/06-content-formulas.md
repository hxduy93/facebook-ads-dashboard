# Skill: 14 Công Thức Viết Content (by Noti)

## Khi nào dùng

Khi user yêu cầu "viết content/copy/quảng cáo/ads" — content bán hàng, không phải blog thông tin.

## 14 công thức

### 1. AIDA — Awareness/Interest/Desire/Action
Mở: gây chú ý → giới thiệu lợi ích → khao khát → CTA

### 2. PAS — Problem/Agitation/Solution
Vấn đề → khoét sâu → giải pháp (SP)

### 3. 4Cs — Clear/Concise/Compelling/Credible
Rõ ràng / Súc tích / Hấp dẫn / Đáng tin

### 4. FAB — Features/Advantages/Benefits
Tính năng → Ưu điểm → Lợi ích cảm xúc

### 5. ACC — Awareness/Comprehension/Conviction
Nhận biết → Hiểu rõ → Tin tưởng

### 6. SLAP — Stop/Look/Act/Purchase
Dừng lại → Nhìn → Hành động → Mua

### 7. BAB — Before/After/Bridge
Trước (vấn đề) → Sau (kết quả) → Cầu nối (SP)

### 8. 5W1H — Who/What/When/Where/Why/How
Cho post chi tiết, có data

### 9. Storytelling
Khởi đầu → Xung đột → Cao trào → Giải quyết → Bài học

### 10. SSS — Star/Story/Solution
Nhân vật ngôi sao → Câu chuyện → Giải pháp

### 11. Hook-Value-CTA
Hook 3s → Giá trị thực → CTA rõ

### 12. PPPP — Picture/Promise/Prove/Push
Vẽ tranh → Hứa hẹn → Chứng minh → Đẩy hành động

### 13. Funnel content
Top: Awareness | Mid: Consideration | Bottom: Decision

### 14. COC — Curiosity/Outcome/Confidence
Tò mò → Kết quả → Tự tin (cho quảng cáo low-funnel)

## NLP copywriting tích hợp

- **Anchoring** giá: "Giá gốc 5tr → Hôm nay 1.99tr"
- **Loss aversion**: "Bỏ lỡ ưu đãi này = mất X"
- **Social proof**: "11,856 KH đã chọn"
- **Scarcity**: "Còn 7 sản phẩm"
- **Authority**: "Chuẩn Mỹ", "Đối tác chính hãng"
- **Reciprocity**: "Tặng kèm thẻ nhớ 32GB"

## Kết hợp với brand voice Doscom

- **Doscom**: dùng AIDA, FAB, PAS — focus vào pain point an ninh
- **NOMA**: dùng BAB, Storytelling — focus vào kết quả ngay (xe sạch, đèn pha trong)
